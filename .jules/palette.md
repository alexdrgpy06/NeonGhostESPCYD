## 2024-06-01 - Add immediate visual feedback to physical touchscreens
**Learning:** Immediate visual feedback (e.g., momentarily redrawing with inverted colors using a short delay, ~50ms) on physical touchscreens is critical to prevent users from repeatedly tapping buttons.
**Action:** Always implement momentary visual state changes (like color inversion) when handling physical touch events on hardware interfaces to provide tactile confirmation.
