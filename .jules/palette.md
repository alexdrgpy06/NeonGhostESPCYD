## 2024-05-14 - Immediate Visual Feedback
**Learning:** For physical touch interfaces like ESP32 CYD, users repeatedly tap buttons if there's no immediate visual feedback, causing unexpected side effects.
**Action:** Implement momentary visual feedback by briefly inverting colors or altering drawing states (e.g. using a `drawSingleButton` helper) when buttons are tapped before executing the associated action.
