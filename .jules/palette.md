## 2024-05-24 - Immediate Tactile Feedback on Touchscreens
**Learning:** Users lack confidence when tapping elements on a physical TFT touchscreen if there is no immediate visual response, leading to repeated tapping or confusion about whether the input registered.
**Action:** Always provide momentary visual feedback (e.g., inverted colors and a ~50ms delay) when an interactive element is touched on an embedded display to confirm the action before processing the logic.
