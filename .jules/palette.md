## 2026-06-22 - Add immediate visual feedback to physical buttons
**Learning:** Immediate visual feedback (e.g., momentarily redrawing with inverted colors using a short delay, ~50ms) on physical touchscreens is critical to prevent users from repeatedly tapping buttons.
**Action:** Always implement a brief visual 'pressed' state (e.g., inverted colors) for physical touchscreen buttons to acknowledge touch input instantly.
