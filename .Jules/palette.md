## 2025-03-05 - Immediate Visual Feedback on Physical Touchscreens
**Learning:** Physical touchscreens like the CYD (Cheap Yellow Display) often lack the tactile feedback of physical buttons and standard hover states of desktop interfaces. Without immediate visual feedback that a tap has been registered, users are prone to repeatedly tapping a button, potentially triggering an action multiple times or causing frustration if the system is slow to respond.
**Action:** Always implement immediate visual feedback on interactive elements. A common and effective pattern is "press-wait-release" - instantly changing the button's appearance (e.g., inverted colors), briefly delaying (e.g., 50ms), and then restoring the original appearance before executing the associated action logic. This clearly communicates to the user that their input was received.
