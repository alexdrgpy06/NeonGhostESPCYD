## 2025-02-28 - Immediate Visual Feedback on Physical Touchscreens
**Learning:** Immediate visual feedback (e.g., momentarily redrawing with inverted colors using a short delay) on physical touchscreens is critical to prevent users from repeatedly tapping buttons.
**Action:** Implement 'press-wait-release' interaction patterns for interactive buttons using helper functions like `drawSingleButton` to quickly render pressed states before executing actions.
